# IDM-Projekt  
IDM-Requeststrecke als Lernprojekt mit Python & Streamlit

## 🌐 Live Demo  
👉 https://idm-projekt-bburwv57dgfeajwwjd7qhn.streamlit.app/

---

## 🎯 Projektidee  
Eine Streamlit-App zur strukturierten Erfassung von Anträgen im IDM-Umfeld  
mit automatischer Erstellung eines sauberen Request-Texts.

---

## 💡 Warum dieses Projekt  
Ich arbeite im Support- und Operations-Umfeld mit Schwerpunkt Identity Management (IDM).  
Ziel war ein praxisnahes Lernprojekt, um:

- Python-Kenntnisse praktisch anzuwenden  
- UI-Logik besser zu verstehen  
- typische IDM-Prozesse nachzubilden  
- ein eigenes Lernprojekt aufzubauen

---

## ⚙️ Warum Streamlit  
Streamlit ermöglicht eine schnelle UI-Entwicklung direkt in Python.

Vorteile:

- keine HTML-Kenntnisse notwendig
- keine CSS-Kenntnisse notwendig
- schnelle Prototyp-Entwicklung
- sofort sichtbare Ergebnisse

----------------------------------------------------------------------------------------

## Hinweis

Dieses Projekt ist ein Lern- und Demonstrationsprojekt.

Die Anwendung ist öffentlich erreichbar und enthält aktuell keine
Benutzer-Authentifizierung oder Zugriffsbeschränkung.

Ziel des Projekts ist es, typische IDM-Prozesse technisch zu verstehen
und eine einfache Request-Erstellung mit Python und Streamlit umzusetzen.


## Mögliche Erweiterungen

- Benutzer-Authentifizierung (Login)
- Rollenmodell (Admin / User)
- Speicherung der Requests in einer Datenbank
- Export der Requests (z.B. CSV oder PDF)
